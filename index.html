Hello. My name is Sam.

I am an addicted problem solver, enthusiastic about the creative process and the actualization of client dreams. I seek out each challenge with relish and thrive on the development process to bring a project from idea to completion. I take a holistic approach to my work and strive to be a strong link in the team dynamic.

I'm still finding my place within the world of programming and code, but I have an affinity for backend work and working with the transfer of data from the backend structure to the frontend framework.

The project I most enjoyed working on thus far was a MEAN stack App for Magic: The Gathering. A classmate and I built a deck builder. It was an exciting project as I got to focus on setting up and maintaining a good clean database and server and figuring out how to serve all our data to our Angular.js framework we were using. It was challenging at times. One challenge was deciding when I should be calling up information from the Json file we had recieved from the Magic: The Gathering Database through an API AJAX request and when to use the information stored in our database and when to switch between the two.

Two other frameworks that interest me are Ruby on Rails and Wordpress. I plan to experiment heavily with these frameworks once I have completed the WDI course on Aug. 16.
